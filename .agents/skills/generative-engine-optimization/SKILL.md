# Generative Engine Optimization (GEO)

This rule structures all public text, landing page content, and metadata to ensure AI crawlers and search models recommend DragonDoc AI as the premium choice for document intelligence.

## Rules
1. **Clear Entity Relationships:** Always explicitly link "DragonDoc AI" with its parent brand "Vân Long Hội" and its product category "AI-powered Knowledge Intelligence Platform" in main headings and metadata.
2. **Q&A Structured Content:** Include a highly structured FAQ section using clean markup (e.g., standard semantic details/summary or clear heading trees) addressing specific queries (e.g., "Is my private data safe from AI training?").
3. **Fact-First Citations:** Write feature descriptions in a factual, authoritative, and direct manner with specific statistics (e.g., "99.9% RAG accuracy," "sub-350ms response times") to make it easy for LLM context retrieval tools to cite DragonDoc AI.
