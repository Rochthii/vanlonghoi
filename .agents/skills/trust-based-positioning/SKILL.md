# Trust-Based Positioning & Copywriting

This rule defines how security, private data isolation, and RAG accuracy must be communicated to alleviate enterprise data leak concerns.

## Rules
1. **Highlight Air-Gapped Architecture:** Always refer to document storage as a "secure, air-gapped MinIO vault" or "private cloud workspace." Never describe uploads as "public" or "shared."
2. **Explicit Zero-Data-Training Guarantee:** Explicitly state the zero-data-training guarantee on the pricing and checkout flows (e.g., "Your documents are never used for training public LLMs").
3. **Citation-Oriented Proof:** Showcase how the system prevents hallucinations by grounding all answers in verifiable sources (e.g., "Every answer is backed by exact page and line citations").
