# SaaS Copywriting Framework (AIDA)

This rule defines how to write marketing copy for the DragonDoc AI landing page and feature descriptions to maximize user conversion.

## Rules
1. **Attention (Hook):** Start with an immediate, high-impact value proposition. Avoid generic slogans like "AI for documents." Use concrete action statements (e.g., "Transform Static Documents into Living Institutional Wisdom").
2. **Interest (Problem Alignment):** Identify the friction points immediately. Highlight lost productivity (e.g., "spending 2.5 hours per day searching for files") to engage the target audience.
3. **Desire (Solution Benefit):** Present benefits over features. Instead of just listing "RAG architecture," explain what it achieves (e.g., "Zero hallucination search with sub-second retrieval times").
4. **Action (CTA):** CTA labels must be highly active, descriptive, and low-friction (e.g., "Unlock Your Knowledge Guardian," "Explore Interactive Demo").
